# コード整形・コメント規約（日本語テンプレート）

以下はプロジェクトで推奨するファイルヘッダとコメントスタイルのテンプレートです。

ファイルヘッダ（例）
```
/**********************************************************************************
* @author    : 淵脇 未来.
* @date      : YYYY/MM/DD
* @brief     : ファイルの目的を短く（1〜2行）
**********************************************************************************/
```

クラス
```
/**********************************************************************************
* @brief : クラスの役割と責務を短く.
* @note  : 特記事項（所有権やスレッド制約など）があれば追記.
**********************************************************************************/
```

関数（例）
```
/**
 * @brief 処理の概要（1行）
 * @param[in]  paramName 説明（入力）
 * @param[out] outName   説明（出力）
 * @return 処理結果の意味
 */
```

その他
- TODO / FIXME / NOTE は大文字で統一。
- 日付は `YYYY/MM/DD` を使用。
